# Declaraciones de privacidad para tiendas

Estas respuestas corresponden a la versión local actual de PromApp. Deben revisarse si se habilita el backend, cuentas, sincronización, analítica, publicidad o notificaciones.

## Google Play — Data Safety

- ¿La aplicación recopila o comparte datos requeridos por la sección Data Safety?: **No**.
- ¿Los datos del usuario salen del dispositivo mediante PromApp?: **No**.
- ¿Incluye anuncios?: **No**.
- ¿Permite solicitar eliminación?: **Sí, localmente desde Ajustes > Borrar todos los datos**.
- Política de privacidad: usar la versión pública de `PRIVACY_POLICY.md`.

Las notas, ramos y respaldos seleccionados por el usuario se procesan localmente. La exportación de un respaldo es iniciada expresamente por el usuario y el destino queda bajo su control.

## App Store — App Privacy

- Selección sugerida: **Data Not Collected / Datos no recopilados**.
- Tracking / seguimiento: **No**.
- Publicidad de terceros: **No**.
- Vinculación de datos con identidad: **No aplica**.

## Permisos

La versión móvil no solicita cámara, ubicación, contactos, micrófono, fotos, Bluetooth ni notificaciones. Android release no declara permiso de Internet.
